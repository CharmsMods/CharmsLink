# Manual Test Checklist

1. Upload a GLB model and confirm it renders as glass.
2. Adjust Opacity (0–255) and verify the readouts and visual change.
3. Load a texture, adjust Texture Alpha (0–255) and see immediate effect.
4. Verify Samples counter increases during rendering.
5. Pause/Resume preserves the current accumulation.
6. Auto Export triggers once the sample threshold is reached.
7. Export High-Res works and falls back to tiled rendering when needed.
8. Keyboard shortcuts: Space (pause), S (export current), R (reset).
