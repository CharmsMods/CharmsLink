# Plan Files Guide

This folder is now split by intent:

- `Active Plans/`
  - Put only currently live planning documents here.
  - These should describe work that is still being prepared or actively implemented.

- `Archived Plans/`
  - Historical planning, reviews, and superseded implementation notes.
  - Read these only after checking `Important Sources/Whole Site Context.txt` and the code.

## Workflow Rule

When a plan becomes implemented:

1. update `Important Sources/Whole Site Context.txt`
2. keep any valuable idea/history in the original plan file
3. move that plan into `Archived Plans/`

This keeps current-state truth separate from planning history.
